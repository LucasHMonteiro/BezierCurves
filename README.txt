Trabalho 1 De Computação Gráfica: Curva de Bézier Cúbica

------------------------- Compilação -----------------------------------

Execute o arquivo compile.bat (Abra se quiser ver o comando e executar diretamente no terminal)

------------------------- Execução -------------------------------------

O programa inicia com uma janela do terminal aberta esperando uma entrada na forma WxH
onde W é a largura da área desenhável e H a altura.

Ex: 800x600
	W = 800
	H = 600

Clique no interior do retângulo que aparece para definir os pontos desejados
A cada 4 pontos uma curva de Bézier cúbica é desenhada usando estes como pontos de controle

Comandos:

Z : Apaga último ponto desenhado
C : Mostra o convex hull das curvas já desenhadas
P : Mostra a coordenada dos pontos

File > New : Reinicia o programa com um novo sistema de coordenadas
(Vá para a janela do terminal que ainda está aberta para defini-los)

File > Save : Salva configuração atual do programa em um arquivo

File > Open : Abre um arquivo

File > Resize : Modifica o sistema de coordenadas (Vá para a janela do terminal ainda aberta)

File > Exit : Fecha o programa
